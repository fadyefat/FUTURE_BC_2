# Shaco Token (SHACO)

## Overview
**Shaco Token (SHACO)** is a custom ERC-20 token deployed on the Ethereum Sepolia Testnet. This token is designed for learning, testing, and showcasing blockchain development skills. It demonstrates standard ERC-20 functionality along with minting capability.

- **Token Name:** Shaco Token  
- **Symbol:** SHACO  
- **Decimals:** 18  
- **Total Supply:** 1,000,000 SHACO  

---

## Deployment

- **Network:** Ethereum Sepolia Testnet  
- **Contract Address:** `0xF57E05A0F1379eB76C26dd9Dcf4a8CaB312aFD6e`  
- **Initial Supply:** 1,000,000 SHACO tokens  

You can view the contract on [Sepolia Etherscan](https://sepolia.etherscan.io/address/0xF57E05A0F1379eB76C26dd9Dcf4a8CaB312aFD6e).

---

## Features

- Standard **ERC-20** functionality (transfer, approve, balanceOf, allowance)  
- **Minting function** for the owner to issue new tokens  
- Designed for educational purposes and testnet experiments  

---

## How to Add to MetaMask

1. Open MetaMask → **Assets** → **Add Token** → **Custom Token**  
2. Enter the **Contract Address**: `0xF57E05A0F1379eB76C26dd9Dcf4a8CaB312aFD6e`  
3. Name and Symbol will auto-populate (if not, enter manually: Shaco Token, SHACO)  
4. Decimals: 18
